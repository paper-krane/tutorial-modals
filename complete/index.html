Some notes on the JavaScript...

When it comes to creating modals, there are various ways of achieving the same goal. This is my approach. I wanted to make a simple snippet of JavaScript that can be reused and all you have to alter are the toggled class names and some CSS styles so, in reality, this plugin is very much so CSS driven.

What makes this snippet of code so great is you can use this for more than modals. You can use it for off-canvas navigations, dropdown menus, and much more. When developing websites, this was my main go to over and over again. Hope you can make use of this as well!
